# 🐺 Lupino Edition – Adivinhe o Número

## 📌 Sobre o Projeto...  
Adivinhe o Número – Lupino Edition é um jogo interativo retrô onde o jogador deve descobrir um número secreto gerado pelo sistema. A cada tentativa, o mascote Lupino reage com expressões diferentes e o jogo fornece dicas informando se o número inserido é maior ou menor que o número correto.

🔗 **Demonstração Online:** [Clique para jogar](https://thsm0ke.github.io/Adivinhe-o-Numero/)

## 🎯 Funcionalidades
✅ Geração automática de número secreto conforme a dificuldade escolhida (Fácil, Médio, Difícil)
✅ Sistema de tentativas com ícones de corações
✅ Timer regressivo para cada jogada
✅ Reações visuais do mascote Lupino (neutro, bravo)
✅ Pontuação e recorde armazenados no navegador

## 🛠️ Tecnologias Utilizadas
- HTML5 – Estrutura da página
- CSS3 – Estilização retrô e responsiva
- JavaScript (ES6+) – Lógica do jogo, timer, pontuação e manipulação do DOM

## 🚀 Como Jogar
- Escolha a dificuldade no menu suspenso.
- Clique em Novo Jogo para iniciar.
- Digite um número no campo e clique em Chutar.
- O jogo mostrará se o número correto é maior ou menor.
- O mascote Lupino reagirá às suas jogadas e você terá 3 tentativas para acertar.

## 🐺 Quem é Lupino?
Lupino nasceu dentro da série “Kalahari”, uma história intensa e cheia de conflitos, onde ele é o fiel pet do protagonista Smoke.
Apesar de ter surgido em um contexto sombrio, Lupino ganhou vida própria e se tornou um ícone — misturando carisma, estilo retrô e lembranças de jogos do passado.

🎮 No jogo: Lupino reage às suas jogadas, mostrando emoções que dão personalidade às partidas.

🖌️ Na identidade visual: Lupino é a logo oficial do Lupin Studio, representando a essência criativa e nostálgica do projeto.

🌌 Na lore: Ele carrega a memória de onde tudo começou — um jogo simples, mas que inspirou uma história inteira.

Lupino é mais do que um mascote: ele é a ponte entre o universo de Kalahari e os jogos interativos do estúdio, sempre presente como guardião e símbolo.
